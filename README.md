GeschiedenisInformatica
=======================

Geschiedenis van de Informatica (B-KUL-G0K34A)

## Structuur

Gelieve vragen en antwoorden als volgt te structuren.

```tex
\subsection{Titel met Hoofdletters}
\begin{question}
De vraag
\end{question}

\begin{solution}
Het antwoord
\end{solution}
```
